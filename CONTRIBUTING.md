# Contributing to Open Science Reflections

## Purpose of Contributions
Contributions should refine the shared understanding of productive openness, strengthen documentation practices, or expand the collection of examples and templates that illustrate balanced transparency.

## Contribution Types
- Conceptual contributions (definitions, analyses, commentary)
- Practical contributions (minimal reproducible examples, templates)
- Critical reflections (conflicting viewpoints, trade-off discussions)

## Expectations for Contributors
- Be concise and transparent
- Clearly attribute sources and prior ideas
- Avoid sharing sensitive or proprietary information
- Follow a lightweight structure, no heavy frameworks

## Submission Process
- Fork the repository or create a dedicated branch before making changes
- Submit a pull request describing the intent and scope of your contribution
- Write commit messages that briefly summarize your goal and the breadth of modifications
